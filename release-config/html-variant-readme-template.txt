HelpButton.qs HTML Variants
========================
Version: __VERSION__

A Qlik Sense utility that injects a configurable help button into the
application toolbar. Works with both client-managed Qlik Sense Enterprise
on Windows and Qlik Cloud. 

This zip contains the HTML / JS deployment variants loaded via content libraries.

Installation
------------
Detailed configuration options and installation instructions are provided 
at the repository doc link below.

For full documentation visit:
